文本样式

css文本样式有三种：文本装饰、英文大小写转换、文本阴影

text-decoration

值 说明
none 取消下划线
underline 出现下划线
overline 在文本头部增加一条上划线
line-through 在文本中部增加一条删除线 
p{
	text-decoration: underline;		 //让文本的底部出现一条下划线
}                                                       
p{
	text-decoration:none;			 //取消下划线
}

text-transform

p{
	text-transform: uppercase;		//英文转换成为大写字母
}

text-shadow

给文本添加阴影：水平偏移，垂直偏移，阴影模糊度（可选），阴影颜色（可选）

p{
	text-shadow:5px 5px 3px black;
}

文本控制

text-align

值 说明
left 文本左对齐，默认
right 文本右对齐
center 文本居中对齐
justify 内容两端对齐
start 让文本处于开始的边界
end 让文本处于结束的边界
  • start 和 end 属于css3新增的功能，但是IE和Opera 尚未支持
p{
	text-align:center;
}

white-space

值 说明
normal 空白符被压缩，文本自动换行，默认值
nowrap 空白符被压缩，文本不换行
pre 空白符被保留，遇到换行符会换行
pre-line 空白符被压缩，文本在排满或遇换行符换行
pre-wrap 空白符被保留，文本在排满或遇换行符换行 
p{
	white-space:nowrap;		//设置排版方式（默认设置，多个空白符只能压缩成一个）
}

letter-spacing

值 说明
normal 设置默认间距
长度值 如：”数字” + px 
p{
	letter-spacing:4px;		//设置文本之间的间距
}

word-spacing

值 说明
normal 设置默认间距
长度值 如：”数字” + px 
p{
	word-spacing:14px;		//设置英文单词之间的间距
}

line-height

值 说明
normal 设置默认间距
长度值 如：”数字” + px
数值 如：1,2,3(1,2,3代表的是100%,200%,300% …)
% 如：200% 
p{
	line-height:200%;		//设置段落行高
}

word-wrap

值 说明
normal 单词不断开
break-word 单词断开 
p{
    word-wrap:break-word; 	//让过长的英文单词断开
}

text-indent

值 说明
normal 设置默认间距
长度值 如：”数字” + px 
p{
    text-indent:20px;		//设置文本首行缩进距离
}

text-overflow

text-overflow属性来控制文本的溢出部分，它的作用是对溢出的部分裁剪掉，然后判定是否添加省略号

值 说明
clip 默认值，裁剪文本时不添加省略号
ellipsis 裁剪文本时添加”……”号 
p{
	width:160px;
	white-space:nowrap;
	overflow:hidden;
	text-overflow:ellipsis;
}
