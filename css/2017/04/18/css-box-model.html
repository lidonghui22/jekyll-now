css盒子模型

lidonghui 18 Apr 2017

CSS 盒模型中最基础的就是设置一个元素的尺寸大小

元素尺寸

设置元素的尺寸有三组样式：

属性 值 说明 CSS 版本
width auto、长度值或百分比 设置元素的宽度 1
height auto、长度值或百分比 设置元素的高度 1
min-width auto、长度值或百分比 设置元素最小宽度 2
min-height auto、长度值或百分比 设置元素最小高度 2
max-width auto、长度值或百分比 设置元素最大宽度 2
max-height auto、长度值或百分比 设置元素最大高度 2 
/*设置元素尺寸*/
div {
    width:200px;
    height:200px;
}
/*限制元素尺寸*/
div {
    min-width:200px;
    min-height:200px;
    max-width:800px;
    max-height:800px;
}
/*自适应，auto是默认值，width在auto下是100%的值，height在auto下是自适应*/
div {
    width:auto;
    height:auto;
}

百分比

<div class="a1">
    <div class="a2">我是a1的子元素</div>
</div>

.a1 {
    width:200px;
    height:200px;
}
/*百分比就是相对于父元素来决定的*/
.a2 {
    width:80%;
    height:80%;
}

元素内边距

css盒子模型中，设置元素内部边缘填充空白的大小，称为内边距

属性 值 说明 CSS 版本
padding-top 长度值或百分比 设置顶部内边距 1
padding-bottom 长度值或百分比 设置底部内边距 1
padding-left 长度值或百分比 设置左边内边距 1
padding-right 长度值或百分比 设置右边内边距 1
padding 1 ~ 4个长度值或百分比 简写属性 1

元素外边距

css盒子模型中，设置元素外部边缘填充空白的大小，成为外边距

属性 值 说明 CSS 版本
margin-top 长度值或百分比 设置顶部内边距 1
margin-bottom 长度值或百分比 设置底部内边距 1
margin-left 长度值或百分比 设置左边内边距 1
margin-right 长度值或百分比 设置右边内边距 1
margin 1~ 4长度值或百分比 简写属性 1 
div {
    margin-top:10px;
    margin-bottom:10px;
    margin-left:10px;
    margin-right:10px;
}

溢出处理

当设置了元素固定尺寸且内容过大时，就会出现溢出的问题，溢出主要有两个方向：右侧和底部，可以通过overflow系列样式来控制

属性 值 说明 CSS 版本
overflow-x 溢出值 设置水平方向的溢出 3  
overflow-y 溢出值 设置垂直方向的溢出 3  
overflow 溢出值 简写属性 2  

溢出处理的四种溢出值：

值 说明
visible 默认值,不管是否溢出,都显示内容
auto 浏览器自行处理溢出内容。如果有溢出内容,就显示滚动条,否则就不显示滚动条
hidden 如果有溢出的内容，直接剪掉
scroll 不管是否溢出,都会出现滚动条,但不同平台和浏览器显示方式不同 
/*设置右侧溢出处理方式为自适应（超出范围出现滚动条）*/
div {
    overflow-x:auto;
}

元素盒类型

css盒子模型中display属性可以更改本身盒子类型，盒类型主要分为：

  1. 块级元素（区块）
  2. 行内元素（内联）
  3. 行内-块级元素（内联块）
  4. 隐藏元素
属性 值 说明 css版本
display block 盒子为块级元素 1
display inline 盒子为行内元素 1
display inline-block 盒子为行内-块元素 2
display none 盒子不可见，不占位 1

1.块级元素 解析：能够设置元素尺寸，隔离其他元素功能的元素，如div,p等等

2.行内元素 解析：不能设置元素尺寸，它只能自适应内容，无法隔离其他元素，其他元素会紧随其后，如span，b等

3.行内-块级元素 解析：可以设置尺寸，但无法隔离其他元素的元素，如img

4.隐藏元素 解析：元素隐藏，不在dom文档流中占位

/*块级元素转化成行内元素*/
div {
    width:200px;
    height:200px;
    display:inline;
}
/*行内元素转化成块级元素*/
span {
    width:200px;
    height:200px;
    display:block;
}
/*块级元素转化成行内-块级元素*/
div {
    width:200px;
    height:200px;
    display:inline-block;
}
/*隐藏元素而不占位*/
div {
    display:none;
}

元素浮动

css盒子模型中有一种叫浮动盒，就是通过float属性建立盒子的浮动方向

属性 值 说明 css版本
float left 浮动元素靠左 1
float right 浮动元素靠右 1
float none 禁用浮动 1 
#a {
        background:orange;
        float:left;
}
#b {
        background:pink;
        float:left;
}
#c {
        background:blue;
        float:right;
}

浮动会出现一个问题，当一个盒子被浮动后，紧接着的元素会被干扰，自动堆叠在浮动元素下面，这时可以用clear属性来处理

属性 值 说明 css版本
clear left 左边界不得浮动 1
clear right 右边界不得浮动 1
clear both 两边都不得浮动 1 
/*不管左边还是右边，都清除浮动*/
#c {
    background:blue;
    clear:both;
}
