设置背景

css背景设置样式表如下：

属性 值 说明 CSS 版本
background-color 颜色 背景的颜色 1
background-image none或url 背景的图片 1/3
background-repeat 样式名称 背景图片的样式 1/3
background-size 长度值或其他 背景图像的尺寸 3
background-position 位置坐标 背景图像的位置 1
background-attachment 滚动方式 背景图片的滚动 1/3
background-clip 裁剪方式 背景图片的裁剪 3
background-origin 位置坐标 背景图片起始点 3
background 复合值 背景图片简写方式 1

background-color

值 说明 CSS 版本
颜色 设置背景颜色为指定色 1
transparent 设置背景颜色为透明色 1 
div {
    background-color:red;
}

background-image

值 说明 CSS 版本
none 取消背景图片的设置 1
url 通过URL设置背景图片 1 
/*url是图片路径，设置整个页面以这个图片为背景图，如果图片太小，则会出现多张进行填充*/
body {
    background-image:url(loading.png);
}

background-repeat

值 说明 CSS 版本
repeat-x 水平方向平铺图像 1
repeat-y 垂直方向平铺图像 1
repeat 水平和垂直方向同时平铺图像 1
no-repeat 禁止平铺图像 1 
/*整个页面以这个图片为背景，如果图片太小也禁止平铺*/
body {
    background-image:url(loading.png);
    background-repeat:no-repeat;
}

background-position

值 说明 CSS 版本
top 将背景图片定位到元素顶部 1
left 将背景图片定位到元素左部 1
right 将背景图片定位到元素右部 1
bottom 将背景图片定位到元素底部 1
center 将背景图片定位到元素中部 1
长度值 使用长度值偏移图片的位置 1
百分数 使用百分数偏移图片的位置 1 
/*将背景图片置于页面上方，如果想置于左上方则值为：top left*/
body {
    background-image:url(loading.png);
    background-repeat:no-repeat;
    background-possition:top;
}

/*将背景图相对于父元素偏移左边10px，上边20px;*/
body {
    background-image:url(loading.png);
    background-repeat:no-repeat;
    background-possition:10px 20px;
}

background-size

值 说明 CSS 版本
auto 默认值，图像以本尺寸显示 3
cover 等比例缩放图像，使图像至少覆盖容器，但有可能超出容器 3
contain 等比例缩放图像，使其宽度、高度中较大者与容器横向或纵向重合 3
长度值 CSS 长度值，比如 px、em 3
百分数 比如：100% 3 
/*使用cover相对于100%，全铺一张大图，这个非常实用*/
body {
    background-image:url(loading.png);
    background-size:cover;
}
/*尽可能让图片完整显示在元素内*/
body {
    background-image:url(loading.png);
    background-size:contain;
}
/*长度的用法，分别表示长和高*/
body {
    background-image:url(loading.png);
    background-size:240px 240px;
}

background-attachment

值 说明 CSS 版本
scroll 默认值，背景固定在元素上，不会随着内容一起滚动 1
fixed 背景固定在视窗上，内容滚动时背景不动 1 
/*fixed属性会导致背景产生水印效果，拖动滚动条背景也不会产生移动*/
body {
    background-image:url(loading.png);
    background-attachment:fixed;
}

background-origin

值 说明 CSS 版本
border-box 在元素盒子内部绘制背景 3
padding-box 在内边距盒子内部绘制背景 3
content-box 在内容盒子内部绘制背景 3 
div {
    background-image:url(loading.png);
    background-origin:border-box;   /*背景图从边框以内开始渲染*/
    background-origin:padding-box;  /*背景图会渗透到边框上，从边框位置开始渲染*/
    background-origin:content-box;  /*背景图只在盒子最里面的元素层渲染*/
}

background-clip

值 说明 CSS 版本
border-box 在元素盒子内部裁剪背景 3
padding-box 在内边距盒子内部裁剪背景 3
content-box 在内容盒子内部裁剪背景 3 
div {
    background-image:url(loading.png);
    background-clip:border-box;     /*背景图从边框以内开始裁剪*/
    background-clip:padding-box;    /*背景图会渗透到边框上，从边框位置开始裁剪*/
    background-clip:content-box;    /*背景图只在盒子最里面的元素层裁剪*/
}

background

背景图简写顺序如下: 背景颜色，背景图片，背景图片是否平铺，滚动方式，位置坐标 / 图片尺寸，渲染方式，裁剪方式 （“/”是固定写法，目的是隔开两个属性，这样写才有效）

div {
    background: red url(loading.png) no-repeat scroll left / 100% border-box content-box;
}
