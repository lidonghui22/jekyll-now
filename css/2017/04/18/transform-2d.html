css3 2D变形transform

lidonghui 18 Apr 2017

css3提供了元素变形效果，也叫变换，它可以将元素实现旋转、缩放和平移的功能。

属性值 说明
transform 指定应用的变换功能
transform-origin 指定变换的起点

transform

属性值 说明
none 无变换
translateX()、translateY()、translate() 元素在水平、垂直、或两个方向都进行位移;单位(长度值或百分数值)
scaleX()、scaleY()、scale() 元素在水平、垂直、或两个方向都进行缩放;单位(数值)
rotate() 旋转角度;单位(角度)
skewX()、skewY()、skew() 元素在水平、垂直、或两个方向都倾斜;单位(角度)
matrix() 自定义变换;单位(4~6 数值,逗号隔开)
