使用css

解析：css样式由一条或多条以分号隔开的样式声明组成，每条声明包含一个键值对

<p style="color:red;font-size:50px;">这是一段文字</p>

使用方式

元素内嵌样式

<p style="color:red;font-size:50px;">这是一段文字</p>

文档内嵌样式

<style type="text/css">
p{
color:blue;
font-size:16px;
}
</style>

外部引入样式

<link rel="stylesheet" href="text/css" href="style.css">

层叠和继承

样式层叠：同一个元素通过不同的方式设置样式产生样式的重叠

优先级（从低到高）：

  1. 浏览器自带样式
  2. 外部引入样式
  3. 文档内嵌样式
  4. 元素内嵌样式
    如果要强制设置样式的优先级为最高，使用！important
    color : green	!important

样式继承：某一个子元素得到它父元素的样式

注意：样式继承只适用于元素的外观（文字、颜色、字体等），而元素在页面上的布局样式不会被继承。

<!-- b元素继承了p元素的样式 -->
<p style="color:red">
	<b>html5</b>
</p>
