嵌入元素汇总

元素名称 说明
img 嵌入图片
map 定义客户端分区响应图
area 表示一个用户客户端分区响应图的区域
audio 表示一个音频资源
video 表示一个视频资源
iframe 嵌入一个文档
embed 用插件在 HTML 中嵌入内容
canvas 生成一个动态的图形画布
meter 嵌入数值在许可值范围背景中的图形表示
object 在 HTML 文档中嵌入内容
param 表示将通过 object 元素传递给插件的参数
progress 嵌入目标进展或任务完成情况的图形表示
source 表示媒体资源
svg 表示结构化矢量内容
track 表示媒体的附加轨道（例如字幕）

嵌入元素解析

img嵌入图像

<img src="img.png">
属性名称 说明
src 嵌入图像的 URL
alt 当图片不加载时显示的备用内容
width 定义图片的长度（单位是像素）
height 定义图片的高度（单位是像素）
ismap 创建服务器端分区响应图
usemap 关联map元素

map创建分区响应图

解析：通过图片中的热点进行超链接

<img src="img.png" alt="风景图" width="339" height="229" usemap="#Map"> 
<map name="Map">
    <area shape="rect" coords="31,28,112,100" href="http://www.baidu.com" target="_blank" alt="方形">
    <area shape="circle" coords="187,142,47" href="http://www.google.com" target="_blank" alt="圆形">
    <area shape="poly" coords="287,26,240,66,308,112" href="http://www.soso.com" target="_blank" alt="多边形">
</map>

iframe嵌入另一个html页面

解析：iframe表示内嵌一个html页面，name作用是和上面链接target进行关联，点击哪个链接在iframe中就会显示哪个页面。

<a href="index.html" target="in">index</a> | 
<a href="http://www.baidu.com" target="in">百度</a>
<iframe src="http://www.baidu.com" width="600" height="500" name="in">
</iframe>

embed嵌入插件内容

解析：embed元素是扩展浏览器功能，这里添加了一个土豆网的flash视频，src必填，type表示插入内容的类型，width和height表示显示内容的宽、高

<embed src="http://www.tudou.com/v.swf" type="application/x-shockwave-flash" width="480" height="400"></embed>

object元素

解析：object元素和embed一样，只不过object是html4的标准，而embed是html5标准

progress显示进度

解析：显示进度条，value表示默认显示的值，max表示最大值，一般通过js控制内部的值。

<progress value="30" max="100"></progress>

meter显示范围里的值

解析：10~100的范围，低于40是过低，高于80是过高，最佳值为60，目前值为20。 meter显示某个范围内的值min和max表示范围边界，low表示小于他的值，过低；high表示高于他的值，optimum表示最佳，但是不会出现效果

<meter value="20" min="10" max="100" low="40" high="80" optimum="60"></meter>
