文本元素汇总

元素名称 说明
a 生成超链接
br 强制换行
wbr 可安全换行
b 标记一段文字但不强调
strong 表示重要
i 表示外文或科学术语
em 表示强调
code 表示计算机代码
var 表示程序输出
samp 表示变量
kdb 表示用户输入
abbr 表示缩写
cite 表示其他作品的标题
del 表示被删除的文字
s 表示文字已不再确认
dfn 表示术语定义
mark 表示与另一段上下文有关的内容
q 表示引自他处的内容
rp 与 ruby 元素结合使用，标记括号
rt 与 ruby 元素结合使用，标记括号
ruby 表示位于表意文字上方或右方的注音符号
bdo 控制文字的方向
small 表示小号字体内容
sub 表示下标字体
sup 表示上标字体
time 表示时间或日期
u 标记一段文字但不强调
span 通用元素，没有任何语义。一般配合 CSS 修饰。

文本元素解析

<b>		  表示关键字和产品名称，实际就是加粗        	//<b> html5 </b>
<strong>  表示重要文字，实际就是加粗，和<b>一样       //<strong> html5 </strong>
<br>	  强制换行
<wbr>	  安全换行（遇到很长的内容，如果非要换行，可做个标记让浏览器知道哪个位置可以换行)		//<span>youaremylove<wbr>123456</span>
<i>		  表示倾斜
<em>	  表示强调倾斜，和<i>一样实际就是倾斜
<s>		  删除线
<del>	  表示强调删除线，和<s>一样
<u>		  表示给文字加下划线
<ins>	  表示给文字加下划线，和<u>一样
<small>	  将文本放小一号
<sup>	  上标 ,<sub>下标
<code>	  标示计算机代码片段
<abbr>	  没有实际作用，表示一段文本的缩写
<dfn>	  倾斜，表示术语
<q>		  表示引用，会给包含内容加双引号
<cite>	  引用其他作品的标题，实际效果就是加粗
<ruby>	  语言元素
<bdo>	  设置文字方向，<bdo dir="rtl"> 内容 </bdo>   rtl(文字从右到左)   ltr(文字从左到右)
<mark>    突出显示文本，实际作用就是加上一个黄色背景，黑色文字
<time>	  表示日期和时间
<span>	  表示一般性文本
