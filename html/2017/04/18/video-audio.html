音频和视频

lidonghui 18 Apr 2017

video视频元素

属性名称 说明
src 视频资源的 URL
width 视频宽度
height 视频高度
autoplay 设置后，表示立刻开始播放视频
preload 设置后，表示预先载入视频
controls 设置后，表示显示播放控件
loop 设置后，表示反复播放视频
muted 设置后，表示视频处于静音状态
poster 指定视频数据载入时显示的图片

嵌入一个webm格式视频

解析：video插入视频，主流格式为：webm、mp4、ogg等，src表示路径url、width宽，height高

<video src="test.webm" width="800" height="600"></video>

附加属性

解析：autoplay表示自动播放，controls表示显示播放控件，loop表示循环播放，muted表示静音

<video src="test.webm" width="800" height="600" autoplay controls loop muted></video>

预加载设置

解析：preload有三个值：none表示不加载，metadata表示播放前值加载第一帧画面信息，auto表示请求浏览器尽快下载整个视频

<video src="test.webm" width="800" height="600" controls preload="none"></video>

使用预览图

解析：poster表示视频的预览图

<video src="test.webm" width="800" height="600" controls poster="img.png"></video>

兼容多个浏览器

解析：通过元素引入多种格式的视频，让各浏览器保持兼容

<video width="800" height="600" controls poster="img.png">
	<source src="test.webm">
	<source src="test.mp4">
	<source src="test.ogg">
	<object>这里引入了flash播放器实现IE9以下</object>
</video>

audio音频元素

video的属性值，audio都可以用，比如loop 属性名称 | 说明 —|— src | 视频资源的 URL autoplay | 设置后，表示立刻开始播放音频 preload | 设置后，表示预先载入音频 controls | 设置后，表示显示播放控件

嵌入一个音频

解析：和视频一样

<audio src="test.mp3" controls autoplay></audio>

兼容多个浏览器

解析：和视频一样

<audio controls>
	<source src="test.mp3">
	<source src="test.m4a">
	<source src="test.wav">
</audio>
