表单元素

lidonghui 18 Apr 2017

表单元素汇总

属性名称 说明
form 表示html表单
input 表示用来收集用户输入数据的控件
textarea 表示可以输入多行文本的控件
select 表示可以用来提供一组固定选项
option 表示提供一个选项
optgroup 表示一组相关的option元素
button 表示用来提交或重置的按钮
datalist 定义一组提供给用户的建议值
fieldset 表示一组表单元素
legend 表示fieldset元素的说明性标签
label 表示表单元素的说明标签
output 表示计算结果

表单元素解析

form定义表单

元素名称 说明
action 表单提交的页面
method 表单的请求方式：GET和POST两种，默认GET
enctype 浏览器对发送给服务器的数据所采用的编码格式
name 设置表单名称，方便程序调用
target 提交时的目标位置：_blank, _parent, _self, _top
autocomplete 设置浏览器记住用户输入的数据，实现自动完成表单，有on和off两种，默认为on
novalidate 设置是否执行客户端数据有效性检查

解析：form定义一组表单，method提交方式，action提交到哪个地址，autocomplete=”off”关闭自动提示功能。

<form method="post" action="http://www.baidu.com">
	<button>提交</button>
</form>

input表示用户输入数据

元素名称 说明
autofocus 让光标聚焦在某个input元素上，让用户之间输入
disabled 禁用input元素
autocomplete 单独设置input元素的自动完成功能
form 让表单外的元素和指定的表单挂钩提交
type input元素的类型
name 定义input元素的名称，以便接收值

解析：input元素默认会出现一个单行文本框

<input name="user">

<input name="user" autofocus>            	//光标默认选中input

<input name="user" disabled>            	//禁用input

<input name="user" autocomplete="off">		//禁用输入提示

<form method="get" id="register">
...
</form>

<input name="email" form="register">		//表单外的input进行关联

input中的type

input元素可以生成一个输入文本框，通过不同的属性值，可以限制输入内容

属性名称 说明
text 一个单行文本框，默认值
password 隐藏字符的密码框
search 搜索框，在某些浏览器输入内容时会出现×标记取消
submit、reset、button 生成一个提交、重置、普通按钮
number、range 只能输入数值的框；只能输入在一个数值范围的框
checkbox、radio 复选框，单选框
image、color 生成一个图片按钮；带颜色的按钮
email、tel、url 生成一个检测电子邮件、号码、网址的文本框
date、month、time、week、datetime、datetime-local 获取日期和时间
hidden 生成一个隐藏控件
file 生成一个上传控件
type为text时
属性名称 说明
list 指定为文本框提供建议值的datalist元素，其值为datalist元素的id值
maxlength 设置文本框的最大字符长度
pattern 用于输入验证的正则表达式
placeholder 控件的提示文字
readonly 文本框只读
disabled 文本框禁用
size 设置文本框宽度
value 设置文本框初始值
required 表示用户必须输入一个值，否则验证不通过 
<input type="text" size="50">            		  //设置文本框长度
<input type="text" maxlength="10">            	  //设置文本框输入字符长度
<input type="text" value="初始值">            	  //设置文本框的初始值
<input type="text" placeholder="请输入提示内容">   //设置提示内容
<input type="text" readonly>            		  //设置内容为只读，可以提交数据
<input type="text" disable>            			  //设置为不可用，不能提交数据

<input list="abc">
<datalist id="abc">
	<option value="苹果">苹果</option>            //第一种方法
	<option value="香蕉" label="香蕉">            //第二种方法
	<option value="西瓜">                         //第三种方法
</datalist>                                       //鼠标双击文本框时提供建议值
type为password时
属性名称 说明
maxlength 设置密码框最大字符长度
pattern 对输入进行验证的正则表达式
placeholder 输入密码的提示
readonly 密码框只读
disabled 密码框禁用
size 设置密码框宽度
value 设置密码框初始值
required 表示用户必须输入同一个值

解析：文本框一致，所有输入的内容都会显示星号。

<input type="password">
type为search时

解析：和文本框一致，除了firefox浏览器，其他浏览器都会显示一个叉来取消搜索内容

<input type="search">
type为number、range时
属性名称 说明
list 指定为文本框提供建议值的datalist元素，其值为datalist元素id值
min 设置最小值
max 设置最大值
readonly 设置文本框内容只读
required 表示用户必须输入一个值，否则验证不通过
step 只读上下调节值的步长
value 指定初始值

解析：只能在文本框里面输入数字，不同浏览器显示方式不同

<input type="number">
<input type="range">
<input type="number" step="2" min="10" max="100">        //范围和步长
type为date时

解析：文本框可以获取日期和时间的值，但是浏览器支持度不佳，所以还是图鉴使用jQuery等前端库来实现日历功能

<input type="date">
<input type="month">
<input type="week">
<input type="time">
<input type="datetime">
<input type="datetime-local">
type为color时

解析：实现文本框获取颜色功能，但IE不支持

<input type="color">
type为checkbox、radio时
属性名称 说明
checked 设置复选框、单选框是否为选中状态
required 用户必须勾选，否则无法通过验证
value 设置复选框、单选框选中状态时提交的数据，默认是on 
音乐<input type="checkbox">
体育<input type="checkbox">

<input type="radio" name="sex" value="男">男
<input type="radio" name="sex" value="女">女

<input type="checkbox" name="music" chexked value="1">   //默认勾选，值为1
type为submit、reset、button时

解析：和button元素相同

<input type="submit">
<input type="reset">
<input type="button">
type为image时
属性名称 说明
src 指定要显示图像的url
alt 提供图片的文字说明
width 图片的长度
height 图片的高度
提交额外属性 formaction、formenctype、formmethod、formtarget和formnovalidate

解析：生成一个图片按钮，点击图片就实现提交功能

<input type="image" src="img.png">
type为email、tel、url时

解析：email为电子邮件格式，tel为电话格式，url为网址格式

<input type="email">
<input type="tel">
<input type="url">
type为file时
属性名称 说明
accept 指定接受的mime类型
required 表明用户必须提供一个值，否则无法通过验证

解析：生成一个文件上传控件，用于文件的上传

<input type="file" accept="image/gif，image/jpeg，image/png">

label添加说明标签

解析：label可以关联input，让用户体验更好。

<label for="user">
用户名：<input id="user" name="user">
</label>

fieldset对表单进行编组

解析：fieldset元素可以将表单内容分成各个小模块，比如用户私人信息组，用户公开信息组等

<fieldset>...</fieldset>

legend添加分组说明标签

解析：legend元素给分组加上一个标题

<fieldset>
    <legend>注册表单</legend>
</fieldset>

button添加按钮

值名称 说明
submit 表示按钮的作用是提交表单，默认
reset 表示按钮的作用是重置表单
button 表示一般性按钮，没有其他作用 
<button type="submit">提交</button>
<button type="resset">重置</button>
<button>普通按钮</button>

type为submit时，提供以下属性

属性名称 说明
form 指定按钮关联的表单
formaction 覆盖form元素的action属性
formenctype 覆盖form元素的enctype属性
formmethod 覆盖form元素的method属性
formtarget 覆盖form元素的target属性
formnovalidate 覆盖form元素的novalidate属性 
<button type="submit" form="register">提交</button>      //表单外关联提交

select下拉列表

解析：select下拉列表元素至少包含一个option子元素

<select name="fruit">
	<option value="1">苹果</option>
	<option value="2">橘子</option>
	<option value="3">香蕉</option>
</select>

option子元素和optgroup分组元素还包含以下属性

属性名称 说明
name 设定提交时的名称
disabled 禁用下拉列表
form 将表单外的下拉列表与某个表单挂钩
size 设置下拉列表的高度
multiple 设置是否可以多选
autofocus 获取焦点
required 选择验证，设置之后必须选择才能通过 
<select name="fruit" size="30" multiple>        //设置高度并实现多选
<option value="2" selected>橘子</option>        //默认选中

使用optgroup进行分组，label为分组名称

<optgroup label="水果类">
	<option value="1">苹果</option>
	<option value="2">橘子</option>
	<option value="3">香蕉</option>
</optgroup>
<optgroup label="粗粮类">
	<option value="4">大米</option>
	<option value="5">小米</option>
	<option value="6">玉米</option>
</optgroup>

textarea多行文本框

属性名称 说明
name 设置提交时的名称
form 将表单外的多行文本框与某个表单挂钩
readonly 设置多行文本框只读
disabled 将多行文本框禁用
maxlength 设置最大可输入的字符长度
autofocus 获取焦点
placeholder 设置输入时的提示信息
rows 设置行数
cols 设置列数
wrap 设置是否插入换行符，有soft和hard两种
required 这是必须输入值，否则无法通过验证 
//设置一个大小可变更的多行文本框
<textarea name="content">请留下您的建议！</textarea>
//设置一个固定行高和列宽的文本框，设置插入换行符   
<textarea name="content" cols="30" rows="10" wrap="hard"></textarea>
