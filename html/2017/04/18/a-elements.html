超链接和路径

lidonghui 18 Apr 2017

超链接的属性

属性名称 说明
href 指定元素所指资源的 URL
hreflang 指向的链接资源所使用的语言
media 说明所链接资源用于哪种设备
rel 说明文档与所链接资源的关系类型
target 指定用以打开所链接资源的浏览环境
type 说明所链接资源的 MIME 类型（比如 text/html）

这些属性之中，只有href和target比较常用

href属性

解析：href是必要属性，否则标签就变成空元素了， 属性值是http://开头的url，表示跳转到外部网站。

target属性

属性名称 说明
_blank 在新窗口或标签页中打开
_parent 在父窗框组(frameset)中打开
_self 在当前窗口打开（默认）
_top 在顶层窗口打开文档

解析：常用打开方式

_blank			//新窗口打开
_self			//当前窗口打开

相对与绝对路径

1.绝对路径：直接从file://磁盘符开始的完整路径 例：file://d://html5/index.html

2.相对路径：相对于链接页面而言的另一个页面路径 例：index.htm

3.目录结构语法： 如果在同一个主目录下，有多个子目录层次，那就需要目录结构语法 同一个目录：index.html 或者 ./index.html 在子目录：xxx/index.html 在孙子目录：xxx/xxx/index.html 在父目录：../index.html 在爷爷目录：../../index.html

4.锚点设置 通过属性id或者name实现定位 链接

<a href="#1">第一章</a>
<a href="#2">第二章</a>
<a href="#3">第三章</a>
<a href="#4">第四章</a>
<a href="#5">第五章</a>
