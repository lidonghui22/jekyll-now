for in 循环遍历对象和数组

lidonghui 05 Aug 2015

用for循环进行遍历时，如果是对象，则 i 代表的是属性，如果是数组，则代表的是“键”； 用这种方法可以实现对对象和数组的遍历。

示例：

var obj = {aaa:'a123',bbb:'b123'};
for (var i in obj) {
    console.log(i);                             // aaa     bbb
    console.log(obj [ i ]);                     // a123    b123
}

var obj1 = ['a100' , 'a200' , 'a300'];
for (var i in obj1) {
    console.log(i);                             // 0    1    2
    console.log(obj1 [ i ]);                    // a123   b123
}
